一套完整命令版

假设你已经在 GitHub 上建好了一个空仓库，地址是：

https://github.com/你的用户名/GridMM.git

那么你直接在本地项目目录执行：

git remote remove origin
git remote add origin https://github.com/你的用户名/GridMM.git
git branch -M main
git push -u origin main
