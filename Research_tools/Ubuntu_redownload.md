### 新重装流程（覆盖并扩展）

#### 阶段一：准备工作 (重启电脑前)

1.  **备份数据**：确保您在旧Ubuntu系统中的个人文件（`~/Home` 目录下的）已经备份。**此流程将彻底清除它们** 。
2.  **检查BIOS**：确保您的BIOS显卡设置如您上一张图 (IMG\_2078) 所示，处于**“混合模式” (Hybrid Mode)**。这能保证安装程序正常启动。
3.  **准备临时网络 (最关键！)**：
    * 找一部智能手机和对应的数据线。
    * 在手机上准备好打开 **"USB 共享网络" (USB Tethering)**。
    * **我们必须在安装过程中全程联网**，以确保能下载并安装好您缺失的驱动程序（Wi-Fi, NVIDIA, HDMI）。

#### 阶段二：启动安装程序

1.  **插入启动U盘**：将您准备好的 Ubuntu 22.04.5 启动U盘插入电脑。
2.  **启动到U盘**：
    * 重启电脑，在出现 "LEGION" 标志时，**连续按 `F12`**（或Fn+F12）打开启动菜单。
    * 在菜单中，选择您的U盘（可能显示为 "USB HDD" 或U盘品牌名）。
    * *(这比修改BIOS启动顺序更安全方便 （文档建议修改顺序，但按F12更佳）。)*
3.  **进入试用桌面**：
    * 在黑色的 Grub 菜单中，用方向键选择第一个选项 **"Try or Install Ubuntu"** 。
    * 电脑会加载，然后进入一个临时的 Ubuntu 桌面。

#### 阶段三：安装过程 (联网与驱动)

1.  **【关键】连接临时网络**：
    * **不要**立刻双击 "Install Ubuntu"。
    * 先把您的手机通过数据线连接到电脑。
    * 在手机上，打开**“USB 共享网络”**。
    * 等待几秒钟，查看 Ubuntu 桌面右上角的网络图标，**确认它显示为“有线连接”**。
    * *(您可以打开 Firefox 浏览器访问一下网页，确保网络已通。)*
2.  **开始安装**：
    * 确认网络连通后，双击桌面上的 **"Install Ubuntu"** 图标 。
3.  **语言与键盘**：
    * 选择您的语言（建议英文，后续可改中文 ）和键盘布局。
4.  **【关键】更新与软件**：
    * 在 "Updates and other software" 页面：
    * 选择 **"Normal installation"** (标准安装)。
    * **务必勾选** "Download updates while installing Ubuntu" (安装时下载更新)。
    * **务必勾选** **"Install third-party software for graphics and Wi-Fi hardware..."** (为图形和Wi-Fi硬件安装第三方软件)。
    * *（这是修复您所有驱动问题的核心步骤！)*

#### 阶段四：分区操作 (覆盖并扩展)

这是满足您需求的核心步骤。

1.  **选择 "Something else"**：
    * 在 "Installation type" (安装类型) 页面，拉到最下面，选择 **"Something else"** (其他选项) 。点击 "Continue"。
2.  **删除旧的Ubuntu分区**：
    * 您会看到一个包含所有分区的列表。
    * 找到您**原有**的 Ubuntu 系统分区（它在 "Type" 列会显示为 `ext4`）。
    * 选中这个 `ext4` 分区，点击左下角的 **"-" (减号)** 按钮。
    * *(如果有 `linux-swap` 交换分区，也用同样的方法点 "-" 删除它。)*
3.  **创建新根分区 (覆盖+扩展)**：
    * 删除后，旧的 `ext4` 分区会和您“额外选中”的未分配空间**合并**成一块（或几块）**"free space" (空闲空间)** 。
    * 选中这块**最大的** "free space" 。
    * 点击左下角的 **"+" (加号)** 按钮 。
    * 此时会弹出 "Create partition" (创建分区) 窗口 。
    * **Size (大小)**：**把拉杆拉到最满** ，使用您全部的空闲空间（即 "覆盖" + "扩展" 的总和）。
    * **Type (类型)**：`Primary` (主分区) 。
    * **Location (位置)**：`Beginning of this space` (在此空间起始位置) 。
    * **Use as (用于)**：`Ext4 journaling file system` 。
    * **Mount point (挂载点)**：**选择 `/`** (根目录) 。
    * 点击 "OK"。
4.  **确认启动器位置**：
    * 在分区列表下方，有一个 "Device for boot loader installation" (安装启动引导器的设备) 选项。
    * **通常您不需要动它**，它默认会选择您主硬盘的 EFI 启动项（例如 `/dev/nvme0n1`）。
5.  **开始安装**：
    * 点击 "Install Now"。系统会弹出警告，确认您要对磁盘做的更改（删除旧分区并创建新分区）。点击 "Continue"。

#### 阶段五：完成安装S

1.  **设置时区与用户**：
    * 在地图上选择 `Shanghai` (上海) 。
    * 创建您的用户名和密码 。
2.  **等待安装**：
    * 系统会开始安装。因为您勾选了下载更新和第三方驱动，这个过程会从网上下载几百MB的文件，请确保您的手机网络畅通。
3.  **重启电脑**：
    * 安装完成后，系统会提示您重启 。
    * 按提示拔出您的U盘，然后按回车键重启。

重启后，您的电脑应该会正常进入 Grub 启动菜单，选择 Ubuntu，您将进入一个驱动完整（Wi-Fi、NVIDIA、HDMI 均正常）的新系统。
